# DARK TRAP

Hardcore reaction game where one mistake = death.

## 🎮 About the Game

Dark Trap is a fast-paced skill game where you must survive deadly traps and narrow corridors.

Spikes, moving obstacles, and dangerous zones will test your reaction and precision.  
The further you go, the harder it becomes.

There is no room for mistakes.

## ⚡ Features

- Tight and precise controls  
- Increasing difficulty  
- Deadly traps and obstacles  
- Abilities to survive:
  - Shield
  - Time slow
  - Heal

## 🕹 Controls

### PC:
- W A S D — Movement  
- Shift — Dash  
- 1 / 2 / 3 — Abilities  

### Mobile:
- Left joystick — Movement  
- Right button — Dash  
- Ability buttons — Skills
